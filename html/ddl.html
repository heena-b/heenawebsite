DDL Command
CREATE
SQL create the database or its object(ie table,view,etc).
Syntax
create database databasename;
Example
create database emp;

Example:

 Syntax
create table tablename(colname datatype,colname datatype,...);
Example
create table orders(order_num int,orderdate int,status varchar(255));

Example:
ALTER Command
Alter command is helpful to change or modify the structure of the table.
Syntax
alter table tablename add colname datatype;
Example
alter table orders add comments varchar(255);

Example:

 Syntax
alter table tablename drop column colname;
Example
alter table orders drop column status;

Example:
DROP Command
Drop command helps to delete the object from the database.
Syntax
drop table tablename;
Example
drop table orders;

Example:
TRUNCATE Command
SQL Truncate command helps to remove all records from a table.
Syntax
truncate table tablename;
Example
truncate table orders;
