DCL Command
GRANT Command
GRANT command is helpful to provide priveleges to the database.
Syntax
grant priveleges_names on object to user;
Example
grant select,insert,delete on users to 'name'@'localhost;
REVOKE Command
SQL revoke command is to withdraw the user's access priveleges given by using the GRANT command.
Syntax
revoke priveleges on object from user;
Example
revoke select,insert,delete on users to 'name'@'localhost;
