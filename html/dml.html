DML Command
INSERT Command
SQL insert command is helpful to insert the data into a table.
Syntax
insert into tablename(col1,col2,...)values(value1,value2,...);
Example
insert into orders(order_num,orderdate,comments) values(32457812,12-04-2023,'Nice Product');

UPDATE Command
SQL Update command is helpful to update the existing data in a table.
Syntax
update tablename set colname=value1 where condition;
Example
update orders set orderdate=17-05-2023 where order_num=32457812;

Example:
DELETE Command
SQL Delete command helps to delete the records from a table.
Syntax
delete from tablename where condition;
Example
delete from orders where order_num=32457812;
